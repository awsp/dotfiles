### Software Installation

Polybar Install
```sh
yaourt -S polybar
```

QuteBrowser
```sh
pacman -S qutebrowser

# Optional, install PyOpenGL
sudo pip install pyopengl
```

Bmenu
```sh
yaourt -S bmenu
```

Networking-tools (netstat, etc)
```sh
pacman -S net-tools
```
